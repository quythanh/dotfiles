# quythanh's Neovim configuration

## Folder structure

```
├── init.lua - where it all begins
├── lua/
│   ├── plugins/
│   │   └── configs/ - plugin configs written in lua
│   ├── core/
│   │   ├── theme.lua - theme configurations
│   │   ├── highlights.lua - highlight groups
│   │   ├── options.lua - general neovim options
│   │   └── statusline.lua - simple statusline
│   └── lsp/ - external lsp configs
└── plugin/
    └── packer_compiled.lua - generated by packer.nvim
```
